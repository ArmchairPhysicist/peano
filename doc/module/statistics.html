Return Value real

Description

Returns the covariance of two vectors \(\mathbf{x}\) and \(\mathbf{y}\), $$ \sigma(x,y) =\frac{1}{n-1} \sum_{i=1}^n \left(x_i - \mu_x \right) \left(y_i-\mu_y\right) $$
